# Library Management System (OOP in Python)

A menu-driven Library Management System built using Object-Oriented Programming.

## Features
- Add & search books
- Register members
- Borrow & return books
- Due date & overdue tracking
- JSON-based data persistence

## OOP Concepts Used
- Classes & Objects
- Encapsulation
- Inheritance (extendable)
- File Handling

## How to Run
```bash
python main.py
